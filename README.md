# ModernMart-Ecommerce-
 ModernMart — A responsive e-commerce web app built with React, Redux Toolkit, and React Router featuring product browsing, cart management, and category-based navigation.
 
A modern, responsive e-commerce web application built with React.js and Redux Toolkit.

✨ Features
🏠 Browse products across Home, Men & Women categories
🛍️ Add to Cart with increment/decrement quantity controls
💰 Real-time price calculation and cart total
🗑️ Remove items and clear cart
📱 Fully responsive design for all screen sizes
🎨 Clean UI with smooth hover animations

🔧 Tech Stack

Technology	Purpose
React.js	UI Components
Redux Toolkit	State Management
React Router	Page Navigation
Vite	Build Tool & Dev Server
CSS3	Responsive Styling
